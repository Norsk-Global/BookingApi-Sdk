Norsk Global Booking API

    Description

    This SDK will help you to incorporate Norsk Globals Extensive Booking Api (Norsk Booking Api) into every day applications, business processes, and workflows. Use one of our many SDKs to get started quickly and easily integrate API calls into your applications.

    Norsk Global Booking APIs:

    • Retrieve Scanned Image : Retrieved captured image of your shipment present.

    • Retrieve Shipment Dimensions : Retrieve captured weight/dimensions for multiple shipments.

    • Retrieve Item Dimensions : Retrieve captured weight/dimensions for a shipment items.

    • Track Shipment : Retrieve shipment tracking information.

    • Book Shipment : Book a shipment.

    • Retrieve Shipment Label : Generate a label for a booked shipment.
